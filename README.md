# Phishing Detection & Awareness Report

A professional phishing email analysis and employee awareness report. This project analyzes real phishing email samples, identifies common red flags, classifies risk levels, and provides actionable prevention guidelines for corporate security training programs.

## 📄 Report

The full report is available in PDF format:

- [`Phishing_Detection_Awareness_Report.pdf`](./Phishing_Analysis_and_Awareness_Report.pdf)

## 🛠️ Tools Used

- **Google Admin Toolbox - Messageheader** — Email header parsing and route tracing
- **MxToolbox Email Header Analyzer** — SPF/DKIM/DMARC validation
- **VirusTotal** — URL and file reputation checks
- **Manual Header Analysis** — Received path tracing, IP geolocation

- Executive Summary

- Risk Classification Matrix (Safe / Suspicious / Phishing)

- Organizational Recommendations
